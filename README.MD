# Data Engineering Zoomcamp – End-to-End Project

This repository documents my learning journey through the **DataTalks.Club Data Engineering Zoomcamp**, a hands-on course focused on building real-world data engineering systems.

---

## About the Course

The Data Engineering Zoomcamp is a **free 9-week program** covering the fundamentals of data engineering through practical projects.

It focuses on building an **end-to-end data pipeline** using modern tools and best practices.

- Build real-world projects for my portfolio
- Gain hands-on experience with industry tools
